# 基本类型

## string
字符串类型
```typescript
const str: string = 'TypeScript'
```

## number
数字类型
```typescript
const num: number = 1
```

## boolean
布尔类型
```typescript
const bool: boolean = true
```

## Array\<T\>
数组，TypeScript中数组的元素为同一类型
```typescript
const arr: Array<string> = ['JavaScript', 'TypeScript', 'Dart']
```
也可写为
```typescript
const arr: string[] = ['JavaScript', 'TypeScript', 'Dart']
```
